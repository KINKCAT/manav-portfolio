# Ultra-Premium Minimalist Portfolio

A highly sophisticated, ultra-minimalist, typography-driven personal portfolio built with Next.js, React, Tailwind CSS, Framer Motion, and Three.js. 

This project was carefully designed to prioritize layout, spacing, negative space, and a pure luxury aesthetic (inspired by Apple, Linear, and Vercel) over generic web development themes.

## Features
- **Next.js 16 App Router**: Optimized production builds and server-side rendering.
- **Three.js Luxury Background**: A beautiful drifting star particle system on top of a subtle grid overlay.
- **Pure Black Typography Focus**: Avoids cards, bright gradients, and complex decorations.
- **Framer Motion**: Expensive-feeling, perfectly timed ease-curve animations (`[0.16, 1, 0.3, 1]`).
- **Responsive Layout**: Designed to look perfect across mobile, tablet, and ultra-wide desktops.
- **shadcn/ui**: Accessible and unstyled component architecture.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Next.js (React)
- Tailwind CSS (v4)
- TypeScript
- Framer Motion
- Three.js (@react-three/fiber, @react-three/drei)
- lucide-react (Icons)
